# Sistema de Gestión de Clientes Bancarios
Un sistema sencillo de gestión de clientes bancarios desarrollado en Java.

## Características
-Crear arreglo de clientes: Inicialización de la estructura de datos.

-Agregar nuevos clientes: Registro de usuarios en el sistema.

-Ver clientes: Listado completo de los clientes registrados.

-Buscar cliente por ID: Localización rápida mediante identificador único.

-Eliminar cliente: Remoción de registros de la base de datos.

-Ordenar cuentas: Organización de los registros según criterios específicos.

-Mostrar cuentas de ahorro: Filtrado exclusivo para visualizar cuentas de tipo ahorro.

## Tecnologías Utilizadas
-Java

-Programación Orientada a Objetos (POO)

-NetBeans
